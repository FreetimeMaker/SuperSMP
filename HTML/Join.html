How to Join SuperSMP

To Join SuperSMP on Java Edition, you need to start the Java Edition and click on "Multiplayer". Then click on "Add Server" type as Name "SuperSMP" and type in the IP "supersmp.fun". After that, click on "Done" and you can join the server by clicking on it or you click on "Direct Connection" and type in the IP "supersmp.fun" then click "Join Server" and your Done.
